# 🛒 Customer Segmentation using K-Means

## 📌 Project Overview
This project uses K-Means clustering to group customers based on their income and spending behavior.

## 🚀 Features
- Customer grouping using machine learning
- Interactive dashboard using Streamlit
- Visualization of customer segments
- Business insights for decision making

## 🛠️ Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

## ▶️ How to Run

1. Install libraries:
pip install pandas scikit-learn matplotlib streamlit

2. Run the project:
python -m streamlit run app.py

## 📊 Output
- Segmented customer groups
- Graph visualization
- Insights like high-value and low-value customers

## 💡 Use Case
Helps businesses target customers for marketing and improve sales strategies.
